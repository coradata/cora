# Crosswalk graph derivations

Reserved for graph serializations of the concept corpus — RDF/SKOS (concepts as `skos:Concept`, mappings as typed relations) and a JSON graph for visualization tooling. Empty until the derivation is built.

The authoritative crosswalks are the per-concept YAML files under [`../concepts/`](../concepts/); anything here is a derived projection of those. See [`../taxonomy.md`](../taxonomy.md) for the concept buckets and confidence vocabulary the graph will encode.
