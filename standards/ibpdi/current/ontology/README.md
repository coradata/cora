# IBPDI — ontology (OWL/RDF)

Reserved for the OWL/RDF rendering of IBPDI, derived from the native artifact via CORA's extractor toolchain. Empty until the derivation pipeline ships.

The native artifact under [`../native/`](../native/) is canonical; the normalized field inventory under [`../inventory/`](../inventory/) is the machine-readable substrate crosswalks reference today. This directory will hold the OWL/RDF projection once generated, with a published mapping file showing every derived term.
