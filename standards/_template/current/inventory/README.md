# inventory/

Per-module field inventory YAML files for this standard, one file per module.

See [`docs/field-inventory.md`](../../../../docs/field-inventory.md) for the format, path grammar, and OWL-projection roadmap.

Inventories are produced by the extractor adapters in [`tools/extractors/`](../../../../tools/extractors/) and must validate against [`tools/extractors/schema/inventory.schema.json`](../../../../tools/extractors/schema/inventory.schema.json).
