# bedroom_count

The number of bedrooms in a residential unit. May be fractional in some standards (e.g., "0" for a studio, "1.5" for a one-bedroom-plus-den).

**Aliases:** `bedrooms`, `bedroom_qty`, `num_bedrooms`
